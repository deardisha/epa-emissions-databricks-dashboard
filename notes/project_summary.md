# Project Summary

This project uses Databricks SQL to analyze 2023 EPA emissions data across U.S. states and counties.

The main objective was to understand how emissions are distributed geographically and how emissions compare with population at the county level.

The dashboard includes:

- A map of emissions locations across the continental United States.
- A scatter plot comparing emissions per person and population.
- A donut chart showing the top 10 states' share of total emissions.
- A bar chart ranking the top counties by total emissions.

This project demonstrates beginner-to-intermediate SQL analytics skills, including data cleaning, numeric conversion, aggregation, metric calculation, and dashboard creation.
