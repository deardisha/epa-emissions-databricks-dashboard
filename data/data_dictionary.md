# Data Dictionary

This sample dataset contains selected columns from the 2023 EPA emissions dataset used for the Databricks dashboard.

| Column | Description |
|---|---|
| state_id | Numeric identifier for the U.S. state |
| state_abbr | Two-letter U.S. state abbreviation |
| county_state_name | County name combined with state abbreviation |
| county_id | County FIPS-style identifier |
| county_name | County name |
| latitude | Latitude coordinate for the county/emissions location |
| longitude | Longitude coordinate for the county/emissions location |
| population | County population |
| employment | County employment count |
| doe_climate_zone | Department of Energy climate zone identifier |
| ghg_emissions_mtons_co2e | Greenhouse gas emissions measured in metric tons of CO2e |
