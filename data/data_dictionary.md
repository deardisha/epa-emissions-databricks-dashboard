# Data Dictionary

This dataset contains county-level 2023 emissions, population, energy consumption, and related geographic information used for the EPA Emissions Analytics Dashboard.

| Column | Description |
|---|---|
| state_id | Numeric state identifier |
| state_abbr | U.S. state abbreviation |
| county_state_name | County name with state abbreviation |
| county_id | County FIPS-style identifier |
| county_name | County name |
| latitude | County/location latitude |
| longitude | County/location longitude |
| population | County population |
| employment | County employment count |
| doe_climate_zone | Department of Energy climate zone |
| GHG emissions mtons CO2e | Greenhouse gas emissions in metric tons of CO2e |
| vehicle miles traveled (miles) | Total vehicle miles traveled |
| vehicle miles traveled (miles/capita) | Vehicle miles traveled per person |
| consumption columns | Energy or fuel consumption values by category |
| expenditures columns | Expenditure values in millions by category |
| utility customers | Number of utility customers |
| buildings | Number of buildings |
| area (sq. ft.) | Building area in square feet |
